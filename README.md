#Snake Game
A classic Snake game with a graphical interface built in Python using Tkinter.

📝 Description
A simple yet engaging game where you control a snake, collect food, and grow longer. The game features three difficulty levels and visual score tracking.

🎮 Features
Three difficulty levels: Easy, Medium, Hard (different movement speeds)

Two types of food:

Regular food (green) - increases snake length by 1

Special food (rainbow-colored) - increases snake length by 3

Visual effects: The snake has eyes that look in the direction it's moving

Score tracking: Displays snake length at game end

Keyboard controls: WASD or arrow keys for movement
